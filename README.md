# The Fifth Sip

A premium cinematic cocktail lounge landing page built with modern web technologies.

## Overview
The Fifth Sip is a highly interactive, beautifully designed landing page for a luxury cocktail lounge. It features scroll-triggered animations, smooth text reveals, and a modern aesthetic designed to captivate visitors.

## Features
- **GSAP Animations**: Complex, highly performant scroll animations and text effects.
- **Responsive Design**: Flawless experience across mobile, tablet, and desktop.
- **Modern UI**: Dark mode aesthetic, premium typography, and glassmorphism elements.
- **Interactive Menu**: A dynamic cocktail menu slider.

## Tech Stack
- **Framework**: React (Vite)
- **Styling**: Tailwind CSS
- **Animation**: GSAP (GreenSock Animation Platform)

## Installation & Development

To run this project locally:

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/the-fifth-sip.git
   cd the-fifth-sip
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

## Build & Deployment

To build the project for production:
```bash
npm run build
```

To preview the production build locally:
```bash
npm run preview
```
